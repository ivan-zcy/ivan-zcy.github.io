​ 假如a^b = c，我们已知b和c，如何求a？（其中b <= 200, a <= 1e9, c <=1e100 ）

​ 注意！！c的取值范围是1e100！！！

三.log

​ 关于log的巧妙用法

​ 第一个：例如给出a, b, c, d,如何比较a^b与c^d的大小？

​ 我们只需要求p = b✖️log(a); q = d✖️log(c)

​ 如果p - q的绝对值 <= 1e-6，则两者相等，

​ 如果 p < q ，则a^b < c^d，

​ 否则a^b > c^d
